How to Run a ts file
====================
1. install typescript compiler 
   npm i -g typescript
   tsc abc.ts 
    (OR)
    npx tsc abc.ts
2. once the above command is run, equivalent Javascript file will be generated
  that javascript file can be included in HTML file
